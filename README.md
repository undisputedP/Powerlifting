# The Catalogue — Powerlifting Programs

A dynamic single-page web app featuring a curated catalogue of powerlifting programs. Browse, compare, and select programs tailored to your training goals.

## Live Site

**[https://undisputedp.github.io/Powerlifting/](https://undisputedp.github.io/Powerlifting/)**

## About

The Catalogue brings together a wide range of powerlifting programs — from beginner linear progression to advanced periodization — in one clean, browsable interface. No ads, no sign-ups, just programs.

## Features

- Browse a curated library of powerlifting programs
- Dynamic filtering and program details
- Fully client-side — no backend, works offline after first load
- Mobile-friendly layout

## Tech

Self-contained single-file HTML app. All assets (fonts, scripts, styles) are bundled and compressed inline — no external dependencies at runtime.
